Library REST API Service
REST API сервис для управления библиотечным фондом (книги и патенты) с генерацией Excel-отчетов и миграциями через Liquibase.

Требования:
1)Java: JDK 17+
2)Сборка: Maven
3)База данных: PostgreSQL 14+
4)Среда разработки: Intellij IDEA (или любая другая)

Разворачивание:
1)Скачайте папку library-service — это корень проекта.
2)Создайте базу данных командой: createdb library
3)Настройте подключение к БД в файле application.properties.
4)Запустите проект в IDE. Сервер будет доступен по адресу: http://localhost:8080

Отчет:
Для скачивания Excel-отчета со всеми книгами:
http://localhost:8080/books/report
